## Excel 预览与集成开发计划（阶段一：只读高保真）

- [ ] 后端：新增 `excel-open` IPC  
  - 使用 SheetJS 读取指定 `.xlsx`，返回结构：sheet 列表、每 sheet 的 `cells`（值 + 类型）、`styles`（字体/背景/边框/对齐）、`merges`、`colWidths`、`rowHeights`。  
  - 未支持对象（宏、图表、透视等）保持原样，不写回、不破坏源文件。

- [ ] 前端组件 `ExcelPreview`（只读高保真）  
  - Sheet 标签切换。  
  - 按 `colWidths`/`rowHeights` 渲染网格，支持合并单元格。  
  - 还原字体/字号/粗体/下划线/对齐/背景色/边框。  
  - 滚动查看，默认 100% 缩放，预留 Zoom 控件接口。

- [ ] 数据流接入  
  - 打开 Excel 时调用 `excel-open`，将返回数据传入 `ExcelPreview`。  
  - 预览入口与 Word 预览并存（Tab 或切换），保持 UI 一致感。

- [ ] 性能与降级  
  - 首版不做虚拟滚动，保留接口便于后续“仅渲染可视区”。  
  - 大表提示“预览可能简化”，但不破坏源文件。

- [ ] 基本校验与日志  
  - 打开失败/不支持格式时提示；控制台打印文件名、sheet 数、单元格计数。
